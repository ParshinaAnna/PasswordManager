## ✨ Особенности

| Функция | Описание |
|---------|----------|
| 🔒 **Шифрование** | Все пароли шифруются XOR-шифрованием |
| ⭐ **Скрытие паролей** | Пароли отображаются как `****` (всегда 4 звездочки) |
| 🔑 **Мастер-пароль** | Защита доступа к хранилищу |
| 🔍 **Поиск** | Быстрый поиск по названию сервиса |
| 🎲 **Генератор** | Встроенный генератор надежных паролей (6-32 символов) |
| 📋 **Копирование** | Копирование пароля в буфер обмена одним кликом |
| 💾 **Автосохранение** | Автоматическое сохранение при изменениях |

## 📋 Системные требования

- **Операционная система:** Windows 7/8/10/11
- **.NET Runtime:** 6.0 или выше
- **Место на диске:** ~50 MB

## 🚀 Установка

Способ 1: Скачать релиз
1. Перейдите в раздел [Releases](https://github.com/ВАШ_ПОЛЬЗОВАТЕЛЬ/PasswordManager/releases)
2. Скачайте `диплом.zip`
3. Распакуйте в любую папку
4. Запустите `PasswordManager.exe`

Способ 2: Сборка из исходников
https://github.com/ParshinaAnna/PasswordManager/commit/d7f4d4ef2ce374163ee9841752e8fe8e943c004b

Открытие в Visual Studio
cd PasswordManager
start PasswordManager.sln
